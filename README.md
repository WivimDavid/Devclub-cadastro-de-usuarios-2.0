# Devclub cadastro de usuários

Este projeto foi desenvolvido no modulo de React do curso Fullstack pro do Devclub.

## Tecnologias
- React
- React Router Dom
- React Hooks
- Styled Components
- Axios


## Como executar
Para rodar o projeto, basta acessar o diretório do projeto e rodar o comando `yarn dev`.
